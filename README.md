## Каталог продуктов

Проект представляет собой каталог продуктов, разработанный с использованием Clean Architecture и паттерна CQRS (реализованного через MediatR).

## Технологии
 - **Backend**: .NET 9, Clean Architecture, CQRS/MediatR
 - **Database**: PostgreSQL
 - **Frontend**: HTML/CSS/JavaScript
 - **Infrastructure**: Docker Compose, Nginx

## Запуск проекта
```
git clone https://github.com/petrovskky/product-catalog.git
cd product-catalog
docker compose up -d
```
**Frontend**: http://localhost:3000/ \
**API**: http://localhost:3000/api/
